DB Products Sequelize ORM · mounted at /api/products

GET /api/products

Returns all products stored in the database via Sequelize findAll(). Each item includes id, name, description, price, and quantity.

Public
GET /api/products/:id

Fetch one DB product by its integer primary key using findByPk(). Returns 404 if no row matches that ID.

Public
POST /api/products

Create a new product row using Sequelize create(). name is required. description, price, and quantity are optional and default to their schema values.

Public
PUT /api/products/:id

Update a DB product by ID using Sequelize update(). Only send the fields you want changed — partial updates are fully supported.

Public
DELETE /api/products/:id

Delete a product row by ID using Sequelize destroy(). Returns a success message on delete, 404 if the ID does not exist.

Public

About & Static Data mounted at /api

GET /api/about

Returns metadata about the PrimeNexus platform — app name, version, description, and the poweredby field.

Public
GET /api/about/services

Returns the full services list from the static data.js file on the server.

Public
GET /api/about/products

Returns all static products with full details — id, image, name, rating, priceCents, and keywords.

Public
GET /api/about/productdetail

Returns all static products with only id, image, and name — useful for summary or list views that don't need full product data.

Public
GET /api/about/productdetail/:id

Fetch one static product by its UUID string. Note: this ID is a string, not an integer — find() uses strict string matching.

Public
GET /api/about/products/query

Filter static products via query string. ?search=black filters by name prefix. ?limit=2 caps results. Both params are optional and combinable.
